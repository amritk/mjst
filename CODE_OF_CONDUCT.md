# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project — opening issues, submitting pull requests, or engaging in any project space — you agree to uphold its terms.

## Reporting

To report a concern, please open a confidential report via the [GitHub issue tracker](https://github.com/amritk/mjst/issues) (mark it private if needed) or contact the maintainer through their GitHub profile at [@amritk](https://github.com/amritk).

Reports will be reviewed and investigated promptly and fairly. All maintainers are obligated to respect the privacy and security of anyone reporting an incident.
